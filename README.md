
# Project Structure

The project structure  is divided into 5 apps:

- `accounts` - User accounts and authentication.
- `appointments` - Appointments and scheduling.
- `dashboard` - Dashboard and analytics.
    - This app containst a dashboard for:
        - `Admin` - System overview, user management, analytics
        - `Patient` - Profile, upcoming appointments, history, medical records
        - `Doctor` - Daily schedule, patient consultations, notes
        - `Receptionist` - Appointment management, scheduling, patient check-ins
        
- `medical` - Medical records and notes.
- `scheduling` - Scheduling and appointments.
